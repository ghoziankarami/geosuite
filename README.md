## 🚀 Quick Start (3 Langkah)

### 1. Install WebView2 (sekali saja)
Windows 10/11 biasanya sudah terinstall. Jika belum:
- **Otomatis**: double-click EXE Orebit — installer WebView2 muncul otomatis.
- **Manual**: double-click `MicrosoftEdgeWebview2Setup.exe` → Install → Finish.
- **Download manual**: https://developer.microsoft.com/en-us/microsoft-edge/webview2/

### 2. Jalankan Aplikasi
1. Double-click `Orebit-*.exe` (pilih modul yang Anda beli)
2. **SmartScreen warning**: klik "More info" → "Run anyway" (normal untuk software baru)
3. **Antivirus warning**: klik kanan .exe → Properties → centang "Unblock" → OK

### 3. Aktivasi License
1. EXE pertama buka → muncul dialog aktivasi
2. Buka email konfirmasi order dari orebit.id
3. Copy license key (format `OREBIT3.eyJ...xxx.sIg...yyy`)
4. Paste ke kolom input → klik **Activate**

Aktivasi hanya **sekali per komputer**. Selanjutnya EXE langsung terbuka tanpa tanya lagi.

## 🆘 Troubleshooting

**Q: "Windows protected your PC" (SmartScreen)**
A: Klik "More info" → "Run anyway". Software baru belum ditandatangani Microsoft — normal, aman.

**Q: EXE langsung buka di Chrome/Edge (bukan window app)**
A: WebView2 tidak terinstall. Ikuti langkah 1 di atas. Atau klik kanan `MicrosoftEdgeWebview2Setup.exe` → Run as administrator.

**Q: Aktivasi bilang "Invalid key"**
A: 1) Pastikan copy key UTUH (cek spasi di awal/akhir)
2) Key case-sensitive, copy langsung dari email
3) Cek apakah key untuk produk yang BENAR (key BNDL unlock semua, CORE/ASAY/RSRC unlock 1 saja)
4) Hubungi hello@orebit.id jika masih gagal.

**Q: Mau pakai di Linux / Mac?**
A: EXE ini hanya untuk Windows. Untuk Linux/Mac, gunakan versi web di https://geosuite.orebit.id/try/ (fitur lengkap, butuh browser modern).
